# COMP3607_Jeopardy_Project
Java-based multiplayer Jeopardy game simulation 

How to play:

-Navaigate to "Main.java"
-Click "RUN" on the top right (looks like a play button)
-Indicate file type
-Indicate File Location
-Indicate number of players
-Indicate Player's Name
-Chose answer to questions displayed
-Type "QUIT" to exit game early
